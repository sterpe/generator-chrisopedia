## Installation

### Requirements

- [node](http://nodejs.org/) & [npm](https://npmjs.org/)
- [bower](http://bower.io/)
- [Grunt](http://gruntjs.com/) (`npm instal -g grunt-cli`)

### Setup

1. `npm install`
2. `bower insall`
3. `grunt dev`
3. Open http://dev.<%= name %>/`
